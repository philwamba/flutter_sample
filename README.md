### Prerequisites

- Flutter SDK
- Dart SDK
- Git

## Setup

1. Clone the repository:
```bash
git clone https://github.com/philwamba/flutter_sample.git # or download
```

2. Navigate the project directory:
```bash
cd flutter_sample
```

3. Install the necessary packages:
```bash
flutter pub get
```

4. Run the app:
```bash
flutter run
```
